# admin-scaffolds
基于 vue3 + typescript + element-plus + xw-ui 的后台管理系统的脚手架

## 功能

| 功能     | 描述                                                  |
| -------- | ----------------------------------------------------- |
| 项目规范 | JS规范 、Git 提交规范 、样式规范 、代码格式 、Git检查 |
| 环境     | 开发、测试、正式、mock                                |
| 样式管理 | 全局样式、原子化 CSS、样式隔离                        |
| 路由管理 | 顶部菜单栏、侧边栏                                    |
| 权限管理 | 菜单权限、按钮/组件权限                               |
| 组件库   | JSON Scheme 公共组件库、业务组件                      |
| 插件管理 | 按需引入、自定义插件                                  |
| 国际化   | 本地语言数据、远程语言数据                            |
| 主题管理 |                                                       |
| 开发     | mock数据、页面模板生成                                |
| 测试     |                                                       |
| 构建     | 自定义构建                                            |
| 部署     | GitLab 流水线、自定义脚本                             |
